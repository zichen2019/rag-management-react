# RAG 管理平台 - 知识库管理页面设计稿分析

## 设计稿概览
本设计稿展示了"RAG 管理平台"的**知识库管理**页面。页面采用经典的后台管理系统布局（左侧菜单 + 顶部导航 + 主内容区）。主内容区以白色卡片为容器，包含统计概览、筛选栏、知识库列表、右侧信息面板及底部趋势图。

## 功能模块分析

### 一、顶部导航栏
**布局**：固定在顶部，白色背景，底部浅灰分隔线。
**内容**：
| 位置 | 元素 | 样式/交互 |
|------|------|-----------|
| 左侧 | Logo (六边形) + "RAG 管理平台" | 深色背景区域，白色文字 |
| 左中 | 汉堡菜单 + 面包屑 "首页 / 知识库管理" | "首页"蓝色链接，"知识库管理"灰色当前页 |
| 右侧 | 搜索框 | 占位符 "搜索知识库、数据源或文档"，带搜索图标 |
| 右侧 | 通知铃铛 | 红色徽章 "12" |
| 右侧 | 用户头像 + "admin" | 蓝色圆形头像，下拉箭头 |

### 二、左侧导航菜单
**布局**：固定左侧，深色背景 (#1a2332)。
**菜单项**：
1. **知识库管理** (选中状态，蓝色背景 #1890ff，白色文字图标)
2. 数据源管理 (未选中，灰色)
3. 文档管理 (未选中，灰色)
4. 向量管理 (未选中，灰色)
5. 检索测试 (未选中，灰色)
6. 问答测试 (未选中，灰色)
7. 应用管理 (未选中，灰色)
8. 权限管理 (未选中，灰色)
9. 系统设置 (未选中，灰色)
10. 操作日志 (未选中，灰色)
底部："« 收起菜单"

### 三、主内容区 - 页面标题
**位置**：主内容区顶部左侧。
**内容**：
- **主标题**："知识库管理" (20px, 黑色，加粗)
- **描述**："管理您的知识库，支持知识库的创建、配置和监控" (12px, 灰色)
**右侧操作**：
- **创建知识库按钮**：蓝色背景 (#1890ff)，白色文字 "创建知识库"，加号图标，圆角 4px。

### 四、统计概览模块
**布局**：4 个白色卡片水平排列，圆角 8px，阴影。
**内容**：
1. **知识库总数**：图标 (蓝色)，数字 "12"，"较上月 ↑ 20%" (绿色)
2. **文档总数**：图标 (绿色)，数字 "18,542"，"较上月 ↑ 15.3%" (绿色)
3. **向量总数**：图标 (橙色)，数字 "235.6 万"，"较上月 ↑ 18.7%" (橙色)
4. **存储空间**：图标 (蓝色)，数字 "256.7 GB"，"较上月 ↑ 12.1%" (蓝色)

### 五、主数据区（左右分栏）

#### 5.1 左侧：知识库列表
**布局**：白色卡片，占据左侧约 2/3 宽度。
**筛选栏**：
- **Tab 切换**："全部知识库"(选中，蓝色下划线), "我创建的", "共享给我的"
- **搜索框**：占位符 "搜索知识库名称或描述"
- **下拉筛选**："全部状态", "全部标签"
- **刷新按钮**：循环箭头图标

**数据表格**：
- **表头**：知识库名称，描述，状态，文档数，向量数，存储大小，更新时间，操作
- **行内容**：
  - 知识库名称：图标 + 名称 (如 "公司产品文档")
  - 状态：标签 ("正常"-绿色, "警告"-橙色, "停用"-灰色)
  - 操作：链接 ("查看", "编辑", "更多" + 下拉箭头)
- **分页**：底部右侧，"共 12 条", "< 1 2 >", "10 条/页"

#### 5.2 右侧：信息面板
**布局**：占据右侧约 1/3 宽度，包含 3 个白色卡片。
1. **知识库健康度**：
   - 标题 + "查看详情 >" 链接
   - 环形图 (85 分，良好，绿色)
   - 列表：健康 (10, 83.3%), 警告 (2, 18.7%), 异常 (0, 0%)
2. **数据源类型分布**：
   - 标题 + "查看详情 >" 链接
   - 环形图/饼图 (总数 45)
   - 图例：PDF, Word, Excel, Markdown, 网页，其他
3. **最近上传**：
   - 标题 + "查看全部 >" 链接
   - 列表：文件图标，文件名，知识库归属，时间

### 六、底部趋势模块
**布局**：底部 4 个小型折线图卡片，水平排列。
**内容**：
1. **知识库趋势**：文档数量 (+1,250)，折线图
2. **向量数量**：向量数量 (+85,420)，折线图
3. **检索次数**：检索次数 (+12,560)，折线图
4. **问答次数**：问答次数 (+8,460)，折线图

## 交互细节
- **表格行 Hover**：鼠标悬停时背景色变为浅灰色 (#f5f5f5)。
- **状态标签**：
  - 正常：#52c41a (绿色文字/背景)
  - 警告：#faad14 (橙色)
  - 停用：#d9d9d9 (灰色)
- **操作链接**：蓝色 (#1890ff)，Hover 时加深。
- **创建按钮**：Hover 时背景色加深。

## 整体样式
- **背景色**：#f0f2f5 (浅灰)
- **卡片**：#ffffff，圆角 8px，阴影 0 2px 8px rgba(0,0,0,0.08)
- **主色**：#1890ff
- **字体**：14px 正文，12px 辅助文字
