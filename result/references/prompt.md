# RAG 管理平台 - 知识库管理页面开发提示词

## 项目概述
请使用 React + TypeScript + Vite + Ant Design 技术栈，在现有项目基础上开发"知识库管理"页面。
**重要**：由于当前项目已存在其他页面（如检索测试、问答测试），必须实现完整的路由机制，使得点击左侧菜单能正确切换页面。

## 整体布局
- **左侧菜单**：固定 220px，深色背景。当前选中"知识库管理"。
- **主内容区**：浅灰背景 (#f0f2f5)，包含白色卡片。
- **路由配置**：需在 `App.tsx` 或主布局中配置 `react-router-dom`，路径 `/knowledge-base` 对应本页面。

## 功能模块详细要求

### 1. 页面标题与操作
- **标题**："知识库管理" (20px, 加粗)
- **描述**："管理您的知识库，支持知识库的创建、配置和监控" (12px, 灰色)
- **操作按钮**：右侧"创建知识库" (蓝色 Primary 按钮，加号图标)。

### 2. 统计概览 (Top Cards)
- 4 个卡片水平排列，包含图标、数字、环比趋势。
- 图标颜色区分：蓝、绿、橙、蓝。
- 趋势文字：上升为绿色，数字加粗。

### 3. 筛选与列表区
- **Tab 切换**：全部知识库 (选中), 我创建的，共享给我的。
- **工具栏**：搜索框、状态下拉、标签下拉、刷新按钮。
- **表格**：
  - 列：知识库名称 (含图标), 描述，状态 (Tag), 文档数，向量数，存储大小，更新时间，操作。
  - 操作列：查看、编辑、更多 (Dropdown)。
  - 分页器：底部右侧。

### 4. 右侧信息面板
- **健康度**：环形图 (使用 ECharts 或 Ant Design Charts 或 CSS 模拟)，健康/警告/异常列表。
- **类型分布**：饼图，图例列表。
- **最近上传**：文件列表，含文件类型图标。

### 5. 底部趋势图
- 4 个卡片，含标题、数字、折线图。

## 样式规范
- **主色**：#1890ff
- **卡片**：圆角 8px，阴影
- **状态 Tag**：正常 (绿), 警告 (橙), 停用 (灰)

## 路由与菜单交互 (关键)
- **依赖安装**：若未安装 `react-router-dom`，请执行 `npm install react-router-dom`。
- **路由定义**：
  - `/` -> 重定向到 `/knowledge-base` 或默认页
  - `/knowledge-base` -> 知识库管理页面 (本页面)
  - `/search-test` -> 检索测试页面
  - `/qa-test` -> 问答测试页面
- **菜单联动**：
  - 侧边栏组件需读取当前路由 (`useLocation`)，自动高亮对应菜单项。
  - 点击菜单项应调用 `navigate()` 跳转，而非仅切换 state。
