# RAG 知识库平台 - 检索测试页面设计稿分析

## 设计稿概览

设计稿展示了一个 **RAG 知识库平台** 的 **检索测试** 页面，采用经典的左侧菜单 + 主内容区的布局结构。

**整体布局**：
- 左侧菜单：深色背景，固定宽度约220px
- 主内容区：浅灰色背景（#f0f2f5），包含白色卡片内容
- 主内容区分左右两栏：左栏（查询设置/检索结果）+ 右栏（结果详情）

## 整体样式总结

### 色彩方案
| 用途 | 颜色 | 说明 |
|------|------|------|
| 主色调 | 蓝色（#1890ff 或类似） | 按钮、选中状态、链接 |
| 侧边栏背景 | 深蓝色/深蓝灰色 | #1a2332 或类似 |
| 选中项背景 | 浅蓝色 | 用于高亮当前选中的菜单项和结果项 |
| 标签-技术知识库 | 绿色 | 圆角标签背景 |
| 标签-行业研究报告 | 蓝色 | 圆角标签背景 |
| 文本片段背景 | 浅蓝色 | 用于突出显示匹配文本 |
| 次要文字 | 灰色 | 用于描述、辅助信息 |

### 字体层级
- **页面标题**：~20px，加粗
- **模块标题**：~16px，加粗
- **正文**：~14px
- **辅助文字**：~12px，灰色

### 组件样式
- **卡片**：白色背景（#ffffff），圆角8px，轻微阴影（0 2px 8px rgba(0,0,0,0.08)）
- **按钮**：圆角4px，主按钮蓝色填充，次按钮白色边框
- **标签**：小圆角（2px），彩色背景
- **输入框**：圆角4px，聚焦时蓝色边框
- **分隔线**：浅灰色（#f0f0f0），1px高度

### 间距规范
- **模块标题与内容**：约16px
- **配置项之间**：约16px
- **结果项之间**：约12px（通过分隔线分隔）
- **卡片内边距**：约24px
- **左栏与右栏之间**：约16px
