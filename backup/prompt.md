# RAG 知识库平台 - 检索测试页面开发提示词

## 项目概述
请使用 React + Ant Design 技术栈，开发一个 RAG 知识库平台的检索测试页面。

## 整体布局
- **左侧菜单**：固定宽度约220px，深色背景（#1a2332）
- **顶部导航栏**：固定在页面顶部，横跨整个页面宽度，白色背景，底部有浅灰色分隔线（#f0f0f0）
- **主内容区**：浅灰色背景（#f0f2f5）
- **主内容区分左右两栏**：左栏（查询设置/检索结果）+ 右栏（结果详情），两栏之间间距约16px
- **卡片样式**：白色背景（#ffffff），圆角8px，轻微阴影（0 2px 8px rgba(0,0,0,0.08)），内边距约24px

### 字体层级
- **页面标题**：约20px，加粗，黑色（#262626）
- **模块标题**：约16px，加粗，黑色（#262626）
- **正文**：约14px，黑色（#262626）
- **辅助文字**：约12px，灰色（#8c8c8c）

### 组件样式
- **卡片**：白色背景（#ffffff），圆角8px，阴影 0 2px 8px rgba(0,0,0,0.08)
- **按钮**：圆角4px
  - 主按钮：蓝色背景（#1890ff），白色文字
  - 次按钮：白色背景，灰色边框（#d9d9d9），灰色文字
- **标签**：圆角2px，彩色背景，白色或蓝色文字
- **输入框**：圆角4px，默认灰色边框，聚焦时蓝色边框（#1890ff）
- **分隔线**：浅灰色（#f0f0f0），1px高度

### 间距规范
- **模块标题与内容**：约16px
- **配置项之间**：约16px
- **结果项之间**：约12px（通过分隔线分隔）
- **卡片内边距**：约24px
- **左栏与右栏之间**：约16px
- **行间距**：约8px
