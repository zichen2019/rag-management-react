# RAG 管理平台设计稿分析

## 一、整体布局

### 1.1 布局结构
- **顶部导航栏**：固定高度约56px，白色背景
- **左侧边栏**：宽度约220px，深蓝色背景(#001529)，可收起
- **主内容区**：占据剩余空间，浅灰色背景(#f0f2f5)，内边距24px
  - **上部 - 标题区域**：页面标题、副标题、操作按钮
  - **下部 - 左右布局**：
    - **左侧主内容区**：统计卡片（整行）、知识库列表、趋势图表
    - **右侧面板**：宽度约320px，白色卡片式布局（健康度、数据源分布、最近上传）

## 三、设计规范总结

### 3.1 颜色体系

**主色**：
- 主色：#1890ff（蓝色）
- 成功色：#52c41a（绿色）
- 警告色：#fa8c16（橙色）
- 错误色：#ff4d4f（红色）
- 信息色：#13c2c2（青色）

**背景色**：
- 侧边栏：#001529
- 页面背景：#f0f2f5
- 卡片背景：#ffffff
- 表头背景：#fafafa

**文字色**：
- 标题：#262626
- 正文：#595959
- 辅助：#8c8c8c
- 失效：#bfbfbf
- 链接：#1890ff

### 3.2 字体规范

**字体家族**：-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial

**字号**：
- H1：20px / 600
- H2：16px / 600
- H3：14px / 500
- Body：14px / 400
- Small：12px / 400
- 数值大字：24px / 600

### 3.3 间距系统

- 基础单位：8px
- 组件内边距：16px / 24px
- 组件外边距：16px / 24px
- 卡片间距：16px

### 3.4 圆角规范

- 按钮/输入框：4px
- 卡片：8px
- 头像：50%（圆形）

### 3.5 阴影规范

- 卡片阴影：0 1px 2px rgba(0,0,0,0.03)
- 悬停阴影：0 2px 8px rgba(0,0,0,0.08)
- 下拉阴影：0 3px 6px rgba(0,0,0,0.12)

### 3.6 技术栈推测

基于界面风格判断：
- UI框架：Ant Design 4.x / 5.x
- 图表库：ECharts 或 AntV G2
- 状态标签：Tag组件
- 表格：Table组件（Ant Design）
- 图标：Ant Design Icons
